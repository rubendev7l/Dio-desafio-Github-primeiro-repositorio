## Dicas for me:

[Baixando o Git:](https://git-scm.com/downloads)

- Criar repositório da forma que achar melhor;
- Com botão esquerdo do mouse: Git Bash Here - dentro do repositório que vai colar;
- git clone :arrow_right: colar o link do repositório;
- git status - Ver status 
- git add . ou git add -A - Adiciona a pasta git
- git commit -m " + mensagem"  - Eu commito
- git satus - Eu vou confirmar
- git push origin main - Envio para o GitHub
